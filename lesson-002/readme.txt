At this point, we will be no more simple docker CLI operators.
We are gonna run our containers hosting our code using Nginx
So, let's create our first HTML file

$ mkdir html 
$ cd html

using the "VI" editor

$ vi index.html

now, copy and paste the html file below to the VI editor
after paste the content type :x to exit the VI editor saving the content 
in order to confir wheather the content was property copied type
